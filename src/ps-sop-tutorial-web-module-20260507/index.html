01

PS 是什么 / PS 与 SOP 的区别

PS（Personal Statement）用于展示你的个人经历、成长路径与申请动机；
而 SOP（Statement of Purpose）更侧重学术背景、研究方向与未来规划。

个人陈述（PS）

  • ✓侧重个人经历与成长路径
  • ✓解释申请动机与选择原因
  • ✓展示个人特质与思考方式
  • ✓突出独特性与价值观
VS

目的陈述（SOP）

  • ✓侧重学术背景与研究经历
  • ✓明确研究方向与学习目标
  • ✓展示学术能力与专业潜力
  • ✓强调未来规划与发展路径
核心差异一句话总结： PS 以个人经历为主线，解释申请动机；SOP 以学术经历为基础，明确研究兴趣与未来规划。
02

填写前的重要提示

在开始填写前，先确保你的信息清晰、具体，这将直接影响生成内容的质量。

1. 填写真实且
具体的信息

写清楚做了什么、用了什么方法或工具、取得了什么结果，以及你的具体收获。

2. 避免只写
泛泛的兴趣

不要只写“我感兴趣”，尽量用课程、项目、实习或研究经历来支撑动机。

3. 专业名词建议
使用英文填写

学校、课程、项目、职位、技术工具等名称建议用英文填写，系统识别会更准确。

4. 内容之间
保持一致

技能、项目经历、研究兴趣和申请方向之间，需要形成清晰的对应关系。

5. 优先填写
相关经历

强调与你的申请方向最相关、最能突出你能力与潜力的经历和成果。

填写要具体 → 内容越真实 → 表达越有说服力清晰、真实、相关的信息，是写出高质量 PS 的关键。
